# FBS Tracker – Football Match Score Tracker

## Project Overview
FBS Tracker is a football match score tracking application designed to display live match scores, team details, and player performance information.

## Features
- Live football scores
- Team and squad details
- Match statistics
- Player performance screen
- Interactive UI prototype (Figma)

## Tech Stack (Planned / Used)
- Figma (UI/UX Design – High Fidelity Prototype)
- HTML, CSS, JavaScript (Frontend – planned)
- React.js (planned)
- Node.js & Express.js (planned)
- MongoDB (planned)

## Status
UI/UX high-fidelity design completed.
Development will follow in next phases.
